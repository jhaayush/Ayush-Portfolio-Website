About Me

Ayush designs webpages from scratch, using HTML and CSS. He's also an adept user of Bootstrap as a Front-End technology.

Ayush has proficient knowledge with a host of features of Java, an object-oriented Programming Language. He is able to use this to his advantage, when it comes to understanding and implementing features with websites. One of his favourite frameworks to work with is Angular (his speciality). Projects created using this framework, and also a whole lot other technologies, can be found below.

Ayush has previously worked on many mini-projects using Java, C and C++ through school and college. He was also part of a summer camp as a Teaching Instructor, where he gave students a handle on coding with Object Oriented Programming. These experiences have greatly shaped who he is today, and how he tackles coding problems.

This website is created with the hope that it'll be pleasant to visit, and the projects done would be enjoyable.
